# ConvertX – Live Currency Converter 🚀

**ConvertX** is a premium, real-time currency conversion platform designed for accuracy and elegance. Built with a modern glassmorphic aesthetic, it provides users with instant exchange rates, historical trend analysis, and a seamless cross-device experience.

![ConvertX Banner](https://raw.githubusercontent.com/naveenkumar752/convertx/main/public/logo.svg)

## ✨ Key Features

- ⚡ **Real-Time Rates**: Get the most accurate mid-market exchange rates instantly.
- 📉 **Historical Insights**: Interactive charts powered by `Recharts` to track currency performance over time.
- 🌓 **Smart Dark Mode**: Persistent theme settings that remember your preference across sessions.
- 📰 **Market Insights**: Direct integration with global financial news for informed decision-making.
- 📱 **Responsive Design**: A fluid, "mobile-first" UI that looks stunning on any screen.
- 💎 **Glassmorphic UI**: High-end design featuring aurora gradients and smooth micro-animations.
- 🚀 **PWA Ready**: Installable on your device for quick access.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Charts**: [Recharts](https://recharts.org/)
- **Data Source**: [Frankfurter API](https://www.frankfurter.app/)
- **Icons**: Custom SVG & Lucide

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/naveenkumar752/convertx.git
   cd convertx
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
